Adquisición de ocho equipos computadoras de escritorio para la distrital de atocha

En el departamento de Potosi, Caja Nacional De Salud (Regional Potosi) ha publicado el 3 de Agosto de 2018 la convocatoria 18-0417-08-867504-1-1 para

“Adquisición de ocho equipos computadoras de escritorio para la distrital de atocha”

En las especificaciones técnicas para las computadoras, se lista "Sistema operativo: El equipo debe incluir licencia de sistema operativo Microsoft Windows 10 Profesional 64 bits original en español o versión superior (la licencia deberá validarse o activarse a través de Internet o página web de Microsoft)."
